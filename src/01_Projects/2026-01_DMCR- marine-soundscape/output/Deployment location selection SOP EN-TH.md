# Deployment location selection SOP (EN/TH) — DMCR shallow-reef hydrophone rigs

## 0) Scope and references

**Scope:** Diver-assisted deployment of a bottom-mounted tripod/lander hydrophone rig for shallow reef passive acoustic monitoring (PAM) under DMCR_Soundscape2026.

**References (methodology + evidence):**

- Methodology: [[../task/Hydrophone rig design.md|Hydrophone rig design]]

# Part A — English

## A1) Objective

Select deployment locations that maximize:

1) **Acoustic quality** (minimize flow/wave/vessel contamination; avoid frequency-band loss and clipping)
2) **Scientific representativeness** (choose whether the sample represents a micro-habitat or an integrated reef chorus)

## A2) Roles

- **PI / Acoustic lead:** defines representativeness target (micro-habitat vs integrated chorus), approves final site.
- **Dive lead:** diver safety, underwater placement decisions, environmental protection.
- **Divers (2-person team):** habitat assessment, rig placement, metadata collection.
- **Boat crew:** GPS/position logging, timing, surface safety.

## A3) Required equipment (minimum)

- Rig + recorder + hydrophone + spare fasteners
- Measuring tape or marked line (for height verification)
- Compass (orientation/bearing)
- Slate / waterproof checklist
- Camera (site photos + rig placement photos)
- GPS logging on boat

## A4) Pre-deployment briefing (5–10 minutes)

1) Confirm the **representativeness target**:
   - **Micro-habitat** (record a specific patch type)
   - **Integrated chorus** (avoid nearest-source dominance)

2) Confirm the intended **height band** and orientation options (see [[Deployment configuration factors.md|Deployment configuration factors]]).

3) Agree on the **spacing regime** (single rig vs multiple rigs):
   - Independence goal vs micro-habitat contrast goal.

## A5) Site reconnaissance (in water)

### A5.1 Visual habitat scan (diver)

For each candidate spot, record:

- Substrate (sand / rubble / rock / live coral)
- Topography category (reef wall / reef top / rubble slope / sand edge / sand gap)
- Biological condition nearby (live vs dead structure; obvious hotspots)
- Continuous noise risks (channelized flow, surge, mooring chains, nearby boat lanes)

### A5.2 Optional “short test recording” (recommended)

If feasible, do a short recording (e.g., 2–5 min) at 1–3 candidate spots to compare:

- Flow noise dominance
- Vessel noise dominance
- Shrimp transient dominance / clipping risk

## A6) Decision rules (field checklist)

### A6.1 Height and vertical placement

- Use an adjustable height setting; do not assume one “magic height” works everywhere.
- Default working band: **~0.6–1.0 m above seabed**, adjusted for coral head height and site conditions.

### A6.2 Orientation

- For reef-edge placements, prefer **horizontal / reef-facing** orientation when directivity bias is a concern.
- Record bearing and mount style.

### A6.3 Proximity to reef structure

- Avoid placing the hydrophone **inside the densest, loudest pocket** if the goal is integrated chorus.
- Avoid placing the rig **deep into open sand** expecting it to represent the reef.

### A6.4 Noise regime

- Avoid channelized high-flow zones when biodiversity indices are important (masking risk).
- When possible, position away from boat lanes.

### A6.5 Spacing (if deploying multiple rigs)

- For independent replicates: treat **≥180 m** as a conservative minimum.
- For micro-habitat contrast: closer spacing is acceptable **only if** sites are intentionally different habitat patches.

## A7) Final placement procedure (diver)

1) Place tripod on stable substrate (avoid live coral contact).
2) Level the rig (adjust feet/legs as needed).
3) Set mast height and hydrophone orientation.
4) Confirm:
   - no wobble / toppling risk
   - hydrophone is not contacting frame in a way that transmits vibration (use isolation)
   - flow shield (if used) is secure
5) Start recording and record exact start time.
6) Collect metadata + photos:
   - site label, GPS, depth, height, distance-to-reef estimate, substrate, topography category
   - wind/sea state, vessel activity

# Part B — ภาษาไทย (Thai)

## B1) วัตถุประสงค์

เลือกจุดติดตั้งเพื่อให้ได้:

1) **คุณภาพสัญญาณเสียง** (ลดการปนเปื้อนจากกระแสน้ำ/คลื่น/เรือ; ลดความเสี่ยงของการหายไปของบางย่านความถี่ และการคลิปของสัญญาณ)
2) **ความเป็นตัวแทนทางวิทยาศาสตร์** (กำหนดให้ชัดเจนว่าเก็บ “ไมโครแฮบิแทต” หรือ “คอรัสรวมของแนวปะการัง”)

## B2) บทบาท

- **หัวหน้าโครงการ/ผู้นำด้านเสียง (PI):** กำหนดเป้าหมายความเป็นตัวแทน และอนุมัติจุดสุดท้าย
- **หัวหน้าทีมดำน้ำ:** ความปลอดภัยและการตัดสินใจการวางอุปกรณ์ใต้น้ำ พร้อมการปกป้องแนวปะการัง
- **นักดำน้ำ (ทีม 2 คน):** สำรวจสภาพแหล่งอาศัย วางขาตั้ง เก็บข้อมูลประกอบ
- **ทีมเรือ:** บันทึกพิกัด GPS เวลา และดูแลความปลอดภัยบนผิวน้ำ

## B3) อุปกรณ์ที่ต้องมี (ขั้นต่ำ)

- ชุดขาตั้ง/แท่นวาง (tripod/lander) + เครื่องบันทึก + ไฮโดรโฟน + อะไหล่
- สายวัด/เชือกมีสเกล (ใช้ตรวจระดับความสูง)
- เข็มทิศ (บันทึกทิศ/มุมหันของไฮโดรโฟน)
- กระดานจดใต้น้ำ (เช็กลิสต์)
- กล้องถ่ายรูป (ถ่ายสภาพพื้นที่และตำแหน่งอุปกรณ์)
- บันทึก GPS บนเรือ

## B4) สรุปแผนก่อนลงน้ำ (5–10 นาที)

1) ยืนยัน **เป้าหมายความเป็นตัวแทน**:
   - **ไมโครแฮบิแทต** (เก็บเฉพาะแพตช์/โซนที่กำหนด)
   - **คอรัสรวม** (หลีกเลี่ยงการถูกครอบงำจากสิ่งมีชีวิตที่อยู่ใกล้ที่สุด)

2) ยืนยันแนวทาง **ความสูงจากพื้นทะเล** และตัวเลือกการหันของไฮโดรโฟน (ดู [[Deployment configuration factors.md|Deployment configuration factors]]).

3) ยืนยันแผน **การเว้นระยะระหว่างจุด** (กรณีมีหลายชุดอุปกรณ์):
   - ต้องการความเป็นอิสระของตัวอย่าง หรือเน้นความแตกต่างของไมโครแฮบิแทต

## B5) สำรวจหน้างาน (ใต้น้ำ)

### B5.1 สำรวจสภาพแหล่งอาศัย (นักดำน้ำ)

สำหรับแต่ละจุด ให้บันทึก:

- ชนิดพื้นท้องทะเล (ทราย/เศษปะการัง/หิน/ปะการังมีชีวิต)
- ประเภทภูมิประเทศ (กำแพงแนวปะการัง/ยอดแนว/ลาดเศษปะการัง/ขอบทราย/ช่องว่างทราย)
- สภาพชีวภาพใกล้เคียง (โครงสร้างมีชีวิต vs ตาย; จุด “ฮอตสปอต”)
- ความเสี่ยงเสียงรบกวนต่อเนื่อง (ร่องน้ำกระแสแรง คลื่นกระแทก โซ่สมอ เส้นทางเรือ)

### B5.2 การบันทึกทดลองสั้น (แนะนำ)

ถ้าเป็นไปได้ ให้บันทึกเสียงสั้น ๆ (เช่น 2–5 นาที) ที่ 1–3 จุด เพื่อเทียบ:

- เสียงจากกระแสน้ำ/การไหล (flow noise)
- เสียงเรือ
- ความเสี่ยงเสียงกุ้งดีด (snapping shrimp) ที่อาจครอบงำ/ทำให้สัญญาณคลิป

## B6) กฎการตัดสินใจ (เช็กลิสต์ภาคสนาม)

### B6.1 ความสูงและตำแหน่งแนวดิ่ง

- ใช้ความสูงที่ปรับได้ ไม่ควรยึดติดกับ “ความสูงเดียวใช้ได้ทุกที่”
- ค่าเริ่มต้นที่ใช้งานได้: **ประมาณ 0.6–1.0 เมตรจากพื้นท้องทะเล** แล้วปรับตามความสูงของหัวปะการังและสภาพพื้นที่

### B6.2 การหันทิศของไฮโดรโฟน

- กรณีวางบริเวณขอบแนวปะการัง ให้พิจารณาการวางแบบ **แนวนอนหันเข้าหาแนว** เมื่อมีความเสี่ยงเรื่อง directivity
- บันทึกทิศ (bearing) และรูปแบบการยึด

### B6.3 ระยะห่างจากโครงสร้างแนวปะการัง

- หากต้องการคอรัสรวม ให้หลีกเลี่ยงการวาง “ในจุดที่ดังที่สุด” ใกล้ไมโครแฮบิแทตที่มีเสียงครอบงำ
- หลีกเลี่ยงการวาง “ลึกเข้าไปในพื้นทรายโล่ง” แล้วคาดหวังว่าเป็นตัวแทนของแนวปะการัง

### B6.4 สภาพเสียงรบกวน

- หลีกเลี่ยงร่องน้ำ/โซนกระแสแรง หากต้องการคำนวณตัวชี้วัดความหลากหลาย (เสี่ยงการ masking)
- ถ้าเลือกได้ ให้หลีกเลี่ยงเส้นทางเรือ

### B6.5 การเว้นระยะ (กรณีหลายชุด)

- ถ้าต้องการตัวอย่างอิสระ: ใช้ **≥180 เมตร** เป็นค่าเว้นระยะขั้นต่ำแบบอนุรักษ์นิยม
- ถ้าต้องการเปรียบเทียบไมโครแฮบิแทต: สามารถวางใกล้กว่านั้นได้ **เฉพาะเมื่อ** จุดมีความแตกต่างของแฮบิแทตอย่างตั้งใจ

## B7) ขั้นตอนการวางอุปกรณ์ (นักดำน้ำ)

1) วางขาตั้งบนพื้นผิวที่มั่นคง (หลีกเลี่ยงสัมผัสปะการังมีชีวิต)
2) ปรับให้ได้ระดับและมั่นคง
3) ตั้งค่าความสูงของเสาและการหันทิศของไฮโดรโฟน
4) ตรวจสอบ:
   - ไม่มีการโยก/เสี่ยงล้ม
   - ไฮโดรโฟนไม่สัมผัสโครงสร้างที่ถ่ายทอดแรงสั่นสะเทือน (ใช้ตัวกันสั่น)
   - อุปกรณ์บังการไหล (ถ้ามี) ยึดแน่น
5) เริ่มบันทึกและบันทึกเวลาที่เริ่ม
6) เก็บข้อมูลประกอบ + รูปถ่าย:
   - รหัสจุด, พิกัด GPS, ความลึก, ความสูงจากพื้นทะเล, ระยะห่างโดยประมาณจากแนว, ชนิดพื้น, ประเภทภูมิประเทศ
   - สภาพลม/คลื่น, กิจกรรมเรือ
