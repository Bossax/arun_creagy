# PM - Report Readiness Board

Purpose:
- ใช้ติดตามว่าแต่ละ section ของรายงานอยู่ในสถานะใด
- แยก “มีไอเดียแล้ว” ออกจาก “เขียนรายงานได้แล้ว”

Status scale:
- **0 Empty** — มีแค่หัวข้อ
- **1 Raw Notes** — มี note / comment / workshop debris แต่ยังไม่เป็น analysis
- **2 Evidence Partial** — มี source หรือ note บางส่วน แต่ยังไม่จัดระบบ
- **3 Analysis Ready** — มี synthesis note ที่พร้อมถูกแปลงเป็น prose
- **4 Draft Ready** — เขียนรายงานได้โดยมี evidence chain ชัด

## 2) Critical reframes that must be protected during writing

1. **ความรุนแรง = บริบท ไม่ใช่พระเอกของระบบ**  
   ตามข้อสรุปประชุมล่าสุด รายงานต้องลดการวางความรุนแรงเป็นศูนย์กลาง และหันไปวางเด็ก/เยาวชน การเรียนรู้ และโอกาสชีวิตเป็นแกนกลาง

2. **เด็ก = agent ของอนาคต ไม่ใช่ beneficiary อย่างเดียว**  
   รายงานต้องแสดง agency ของเยาวชน โดยเฉพาะในมิติการเรียนรู้ เศรษฐกิจสร้างสรรค์ และการกำหนดอนาคตตนเอง

3. **การศึกษา x เศรษฐกิจ = แกน scenario หลัก**  
   แกนนี้เป็น reframe สำคัญจากการประชุมกลุ่ม 5G และควรถูกใช้เป็นโครงหลักของ chapter 3 และ chapter 4

4. **เครือข่ายพื้นที่ = execution capacity**  
   ภาคประชาสังคม ผู้นำศาสนา มหาวิทยาลัย องค์กรท้องถิ่น และภาคธุรกิจ ไม่ใช่เพียง “ผู้เกี่ยวข้อง” แต่เป็นโครงสร้างปฏิบัติการของยุทธศาสตร์
