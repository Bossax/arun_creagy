## 2. Working definition of target youth groups

### 2.1 Core target groups
- เด็กกำพร้า
- เด็กและเยาวชนกลุ่มเปราะบางจากครัวเรือนรายได้น้อย
- เด็กที่อยู่ในระบบปอเนาะ/การศึกษาศาสนา
- เด็กที่เสี่ยงหลุดหรือหลุดจากระบบการศึกษา

### 2.2 Still unresolved
- ขอบเขตอายุที่ใช้จริงในรายงาน
- จะรวมเยาวชนที่อยู่นอก 3 จังหวัดแต่เชื่อมกับเครือข่ายเศรษฐกิจ/การศึกษาในพื้นที่หรือไม่
- จะจัดหมวด “เด็กกำพร้า” แยกจาก “เด็กเปราะบาง” หรือวางเป็นวงซ้อนกัน

Usable claims already visible:
- เด็กกลุ่มเปราะบางยังเข้าถึงการศึกษาได้ “บางส่วนแต่ไม่ทั่วถึง”
- มีอย่างน้อย 3 เส้นทางหลักคือ โรงเรียนปกติ, ปอเนาะ/ศาสนา, และหลุดออกจากระบบ

Confidence:
- **Medium** สำหรับภาพเชิงคุณภาพ
- **Low** สำหรับการใช้เป็น baseline เชิงปริมาณ เพราะยังไม่มีตัวเลขสนับสนุนใน pack นี้

### 3.2 Education access pathways
Usable claims already visible:
- ระบบปอเนาะมีความสอดคล้องกับวิถีชีวิตและได้รับความเชื่อถือจากชุมชน
- แต่การต่อยอดไปสู่อาชีพและการรับรองในระบบยังมีข้อจำกัด
- เด็กที่หลุดจากระบบอาจหลุดเพราะต้องทำงาน ดูแลครอบครัว หรือไม่สามารถปรับตัวกับระบบการศึกษาหลักได้

Implication for report:
- Section นี้ควรเขียนให้เห็นว่า “ปัญหา” ไม่ได้อยู่แค่การมีที่เรียน แต่รวมถึงความสามารถของเส้นทางการเรียนแต่ละแบบในการพาเด็กไปสู่ชีวิตที่มีความมั่นคงและมีศักดิ์ศรี

## 4. Draftable baseline claims

These claims are close to usable in a first analysis note, but still need stronger sourcing if they will be stated as high-confidence findings.

1. เด็กกลุ่มเปราะบางในพื้นที่ไม่ได้เผชิญเพียงข้อจำกัดด้านความปลอดภัย แต่เผชิญ “ข้อจำกัดเชิงเส้นทางชีวิต” ที่เชื่อมโยงกันระหว่างการศึกษา รายได้ครัวเรือน สวัสดิการ และความไว้วางใจต่อรัฐ
2. การเข้าถึงการศึกษาของเด็กไม่ได้มีปัญหาเฉพาะเรื่องจำนวนที่นั่งเรียน แต่มีปัญหาเรื่องความสอดคล้องของรูปแบบการเรียนกับชีวิตจริงและโอกาสในอนาคต
3. เด็กกำพร้าและเด็กเปราะบางจำนวนหนึ่งติดอยู่ในจุดต่อระหว่างสิทธิที่มี “ในนาม” กับการเข้าถึงสิทธิจริง “ในทางปฏิบัติ”

## 5. High-priority gaps still blocking strong writing

### 5.1 Quantitative baseline gap
- จำนวนเด็กกำพร้า/เด็กเปราะบางในพื้นที่
- จำนวนเด็กนอกระบบ
- การเข้าถึงสวัสดิการตามช่วงวัย
- ความต่างระหว่างจังหวัดหรือพื้นที่เมือง/ชนบท

### 5.2 Welfare pathway gap
- เด็กกำพร้าต้องผ่านกระบวนการอะไรบ้างกว่าจะเข้าถึงสิทธิ
- จุดหลุดของระบบอยู่ตรงไหน
- actor ใดเป็นด่านสำคัญที่สุดในทางปฏิบัติ

### 5.3 Lived-experience gap
- ยังไม่มีเสียงเยาวชนหรือกรณีชีวิตแบบเป็นระบบพอสำหรับการทำ persona ที่แข็งแรง
- ยังขาดข้อมูลเรื่อง unmet needs และ aspiration ที่เป็นของเยาวชนเอง

### 5.4 Household economy gap
- ยังไม่มีข้อมูลที่แยกให้ชัดว่าแรงกดดันของครัวเรือนมีรูปแบบใดบ้าง เช่น หนี้ รายได้ตามฤดูกาล การย้ายแรงงาน หรือการดูแลข้ามรุ่น

   - กลุ่มเป้าหมาย
   - เส้นทางการศึกษา

   - ความเสี่ยงหลัก
   - actor ที่เกี่ยวข้อง

2. เติม source เชิงปริมาณหรือ administrative source สำหรับ:
   - เด็กกำพร้า
   - เด็กนอกระบบ
   - การเข้าถึงทุน/สวัสดิการ

3. สกัด “journey map” ของเด็กอย่างน้อย 2–3 แบบ:
   - เด็กในโรงเรียนปกติที่เสี่ยงหลุด
   - เด็กในปอเนาะ
   - เด็กที่หลุดออกจากระบบแล้ว
