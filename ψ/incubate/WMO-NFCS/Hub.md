# Hub — WMO–TMD NFCS (Human Settlement & Security sector)

## Purpose (why this project exists)
Create an NFCS action plan / roadmap for **TMD climate services** targeted at the **human settlement & security sector**, grounded in mandate clarity, baseline assessment (supply vs demand), and a costed implementation plan.

## What “done” looks like (primary outputs)
- **Governance framework**: mandate + roles + operating arrangement for a service value chain.
- **Baseline assessment**: stakeholder matrix + capacity gap diagnosis (WMO baseline method adapted to Thailand).
- **Strategic roadmap + costed action plan**: actions, owners, timeline, and budget-eligibility framing.
- **Workshops**: consultation + validation outputs captured as traceable artifacts.
