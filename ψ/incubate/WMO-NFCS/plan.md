# Plan — WMO–TMD NFCS (Human Settlement & Security sector)

## Goal
Deliver an NFCS action plan report and roadmap that operationalizes the **TMD → urban/security user** climate-service value chain (mandates, governance, baseline gap diagnosis, costed action plan).

## Scope (3 work packages)
Source anchor: [`ψ/incubate/WMO-NFCS/inbox_source/NFCS - Implementation Plan.md`](ψ/incubate/WMO-NFCS/inbox_source/NFCS%20-%20Implementation%20Plan.md)

1) **WP1 — Institutional setup + mandate/policy review**
   - Clarify mandate and authority boundaries (Climate Change Act, TMD Act, master plan).
   - Produce an engagement playbook and governance arrangement candidate.

2) **WP2 — Baseline assessment + consultation**
   - Stakeholder mapping + representative qualification rules (avoid tokenism).
   - Baseline survey adaptation + analysis (supply vs demand).
   - Consultation workshop outputs captured as traceable artifacts.

3) **WP3 — Strategic roadmap + validation**
   - Establish task force mechanism (potentially shared with Urban-Act).
   - Draft roadmap + costed implementation matrix.
   - Validate and finalize (workshop #2 + final report).
