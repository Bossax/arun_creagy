# WMO-NFCS Trigger Log

## Purpose
Record **project-relevant triggers** that cause the WMO–TMD NFCS work to reconsider direction, priorities, or narrative.

Use this as the canonical index of **why something changed**.

## Usage and maintenance
- Append new rows; do **not** delete old entries.
- Prefer one row per meaningful trigger.
- When processed, update **Status** instead of rewriting history.
- Link outward to:
  - deliverables via [`ψ/incubate/WMO-NFCS/WMO-NFCS-Deliverable-Map.md`](ψ/incubate/WMO-NFCS/WMO-NFCS-Deliverable-Map.md)
  - submissions via [`ψ/incubate/WMO-NFCS/WMO-NFCS-Submission-Log.md`](ψ/incubate/WMO-NFCS/WMO-NFCS-Submission-Log.md)
  - change summaries via [`ψ/incubate/WMO-NFCS/WMO-NFCS-Change-Log.md`](ψ/incubate/WMO-NFCS/WMO-NFCS-Change-Log.md)
  - neutral history spine via [`ψ/incubate/WMO-NFCS/output/WMO-NFCS-Project-History-Timeline.md`](ψ/incubate/WMO-NFCS/output/WMO-NFCS-Project-History-Timeline.md)
