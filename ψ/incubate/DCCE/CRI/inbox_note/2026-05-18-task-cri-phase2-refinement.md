# Task: CRI Phase 2 Resilience Indicator Refinement
**Date**: 2026-05-18
**Project**: DCCE Climate Resilience Index (CRI)
**Status**: Pending

## Context
Initial research into DOPA's LPA indicators shows a "granularity wall"—data is aggregated at the provincial level, while CRI Phase 2 requires sub-district/local level indicators. We need to rethink the achievable scope for the "Resilience" part of the index.

## Action Items
1. **Gap Assessment**: Document exactly which indicators are missing raw data from DOPA.
2. **Alternative Sourcing**: Explore if there are alternative data sources (other than LPA) that can serve as proxies for resilience at the local level.
3. **Methodology Pivot**: If raw data is unavailable, propose a methodology for "CRI Phase 2 Lite" or a strategy for using "Impact Data" (DDPM) to infer resilience gaps.

## Goal
Ensure CRI Phase 2 remains technically sound and achievable despite data constraints.
