## Lessons Learned

- In CRDB, “CDM governance” is not an abstract best practice; it’s the concrete mechanism to prevent NCAIF from degenerating into a portal-of-portals. The governance deliverable is a domain ownership + boundary model that enables catalog-first and content-as-asset discipline.
- When integrating with an IT-led data catalog effort, treat their enforced metadata as the baseline contract; CRDB adds semantic organization and domain-specific governance without inventing a competing metadata universe.
- Director endorsement must match the reporting line reality: ask Director Nid to ratify domain ownership and internal consensus for the climate risk/adaptation system, not to command external teams.
