## Lessons Learned

- For Track B, treating the v2 capacity dictionary + CBI pack as a single deliverable unit, with Track B8 evidence wiring and a profile-first demonstrator as explicit next steps, helps prevent the v2 dictionary work from becoming an endless refinement loop.
