# Keep NotebookLM MCP skills narrowly scoped (feed prompt + archive atomic notes).

Keep NotebookLM MCP skills narrowly scoped (feed prompt + archive atomic notes). Put the broader research workflow (question→AI research tools→screening→source selection→NotebookLM upload) in workflow docs and the session plan, not in the skill. This prevents scope creep and keeps the NotebookLM skill deterministic and reusable across session styles.
