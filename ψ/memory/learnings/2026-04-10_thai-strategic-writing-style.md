# Learning: Thai Strategic Writing - Avoiding the "Translation Trap"

**Date**: 2026-04-10
**Tags**: #writing-th #foresight #strategic-reporting #thai-language

## Context
When writing the "Foresight 2590" report for the 3 Southern Border Provinces (Thailand), the original English-influenced draft used a "Impact x Uncertainty pair" structure (e.g., "Education x Economic Resilience") for driver analysis. While standard in Western foresight methodology, the user flagged this as "unnatural" and confusing for a Thai strategic audience.

## Observation
1.  **Pairing Complexity**: Western foresight "pairs" create high mental load in Thai sentence structures because the relationship (x) must be articulated with multiple conjunctions (ซึ่ง, ที่, โดยที่).
2.  **Structural Clarity**: Thai policy documents favor a "List and Attribute" structure over a "Matrix" narrative. Explicitly listing each variable with its own `Impact` and `Uncertainty` bullets is much more effective than embedding those levels in a complex paragraph.
3.  **Terminology Choice**: Strategic concepts should use systemic labels (เงื่อนไขชี้ขาด, พอร์ตโฟลิโอการดำเนินงาน, แรงกดทับเชิงระบบ) to build authority, rather than literal translations of English terms.

## Learning
-   **Always prioritize "Variable Listing" over "Scenario Matrix Pairing"** in the written narrative for Thai reports.
-   **Impact and Uncertainty should be explicit headers/bullets**, not implicit prose.
-   **Thai grammar is more stable when variables are handled in isolation before discussing their systemic intersection.**

## Action for Future Sessions
-   When using the `writing-th` skill, specifically look for "Matrix-style" English concepts and convert them to "List-style" Thai structures.
-   Use explicit levels (ระดับผลกระทบ: สูง) as structured metadata within the document to aid readability.
