# Thailand “forest” (legal/administrative) vs IPCC AFOLU “Forest Land” (inventory): keep axes separate

## Pattern

Thailand sources may define **“forest”** in ways that are not directly usable for IPCC AFOLU land-use classification:

- Thai forestry law can define **“forest”** based on **land tenure** (e.g., land not acquired under the Land Code) rather than canopy/height/area thresholds.
- Thai land-administration / zoning rules can define “permanent forest” using **administrative + physical filters** (forest condition, watershed class, slope, soil suitability), which are essential for policy and enforcement but still require an explicit mapping into IPCC’s six land-use categories.

## Why it matters

If you blend these without guardrails, you can end up:

- Classifying a biophysically forested agricultural system as “Forest Land” when inventory rules would call it Cropland (predominant land use), or
- Treating legally “forest” (tenure-based) as automatically “Forest Land” even when IPCC requires a consistent land-use decision tree.

## Operational rule of thumb (until inventory rules are confirmed)

- Use IPCC AFOLU categories as the **accounting frame**.
- Treat Thai legal/admin constructs as **evidence inputs** for land-use intent/constraints.
- Default perennial crop systems (rubber, shaded coffee) to **Cropland** unless the inventory explicitly classifies them as Forest Land.
- If such systems occur inside conservation/permanent-forest zoning, treat them as **conversion** signals (Forest Land → Cropland) rather than re-labelling them as “forest.”
