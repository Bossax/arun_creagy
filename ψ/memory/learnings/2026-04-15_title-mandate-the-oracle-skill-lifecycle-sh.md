# Mandate: The Oracle Skill Lifecycle (Ship/Install/Execute). Archon (Upstream) on

Mandate: The Oracle Skill Lifecycle (Ship/Install/Execute). Archon (Upstream) only SHIPS the 'Form' (code/assets) into the recipient's nervous system (.gemini/skills/ or .roo/skills/). The recipient (Downstream) INSTALLS the skill within its own context. The agent then EXECUTES the skill repeatedly as needed. Archon MUST NEVER 'Push' execution (like rule injection) into the sibling root; it is a violation of Downstream Autonomy.
