# Mandate: Downstream Autonomy. All shipped skills from Archon must be executed by

Mandate: Downstream Autonomy. All shipped skills from Archon must be executed by the recipient Oracles locally, not by Archon pushing the execution. Archon provides the payload in the inbox; the local Oracle provides the will.
