# Lesson Learned — NCAIF sitemap delivery and gap analysis sequencing

When leadership equates “website = platform,” the sitemap must be treated as a decision surface. The policy‑maker entry point should be visually dominant and tied to decision‑ready products (briefing packs, indicators), while the adaptation‑cycle backbone provides coherence for technical users. A content gap analysis becomes actionable only when it is produced as a **chain of artifacts**: inventory → mapping matrix → summary → report insert. This sequence prevents abstract diagnosis and yields outputs that can be pasted directly into the interim report.
