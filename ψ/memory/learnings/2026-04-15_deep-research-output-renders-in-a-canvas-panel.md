# - Deep Research output renders in a canvas panel, not standard `MESSAGE-CONTENT`

- Deep Research output renders in a canvas panel, not standard `MESSAGE-CONTENT` nodes; extraction must use the “Share & Export → Copy contents” flow or a canvas-aware DOM/clipboard fallback.
- UI automation for Gemini modes requires resilient selector strategies (case-insensitive partial match, aria-label fallback, retries, and logging) to avoid silent failures when the UI shifts.
