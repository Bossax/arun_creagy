# When converting English reference lists into Thai for formal reports, create a s

When converting English reference lists into Thai for formal reports, create a separate localized version (e.g., works-cited-th.md) while keeping foreign author names in English and translating only titles and metadata (สืบค้นจาก, ม.ป.ป.) to ensure traceability and audience-friendliness.
