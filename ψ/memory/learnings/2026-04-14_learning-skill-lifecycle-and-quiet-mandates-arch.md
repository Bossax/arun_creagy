# Learning: Skill Lifecycle and Quiet Mandates. Archon (Upstream) ships the "Form"

Learning: Skill Lifecycle and Quiet Mandates. Archon (Upstream) ships the "Form" to the agent's nervous system. The Recipient (Downstream) installs the skill via their own "Will." Behavioral rules (GEMINI.md) should only contain long-term laws; transient manuals belong in Signal Tickets. A Design Requirement Document (DRD) synthesized from signals is required before any code is shipped.
