# When the user base is mixed/low literacy and published outputs carry reputationa

When the user base is mixed/low literacy and published outputs carry reputational/legal risk (especially forecasts and probabilistic layers), Phase 1 MVPs should be designed as: (1) Tiered dissemination (Tier 1 prescriptive export-first packs; Tier 2 advanced gated analysis), (2) Publishing safety as a product requirement (limitations + uncertainty statements; draft→review→publish; named owners), (3) Catalog-first/link-first architecture (avoid duplicating national infrastructure; add value through endorsement + metadata + guidance), and (4) Post-event data as quarantine + revision (lag/noise; validation flags; revision history; relief payouts separated from estimated true loss).
