# When an interim report is explicitly scheduled for revision after an upcoming pr

When an interim report is explicitly scheduled for revision after an upcoming progress report, treat the progress report as a gate for a new drafting cycle: plan to adjust structure, emphasis, and evidence after that meeting rather than prematurely locking content, and maintain a clear change log so revisions after the gate are traceable to stakeholder feedback.
